# **App Name**: HarmonyForge

## Core Features:

- Song Management Form: A web form for creating new songs and editing existing ones, accommodating fields for song number, author, year, title, and dynamically adding/removing individual choruses.
- Song List View: An organized display of all songs with options to view, edit, or delete each entry.
- Delete Song with Confirmation: Securely remove song entries after a confirmation prompt to prevent accidental data loss.
- Real-time Database Synchronization: Immediate updates to the UI as song data is created, edited, or deleted in the Firestore backend.
- Song Search and Filtering: Enable users to quickly locate specific songs by title, author, or song number.

## Style Guidelines:

- A light color scheme evokes clarity and serenity. The primary color is a deep, contemplative blue (#2946A3), chosen to inspire focus and calm without being overly bold.
- The background color is a very light, desaturated blue (#EEF1F7), providing a clean and expansive canvas for content, maintaining the same hue as the primary color.
- An accent color of vibrant blue-cyan (#4AADE0) will highlight interactive elements and call-to-action buttons, creating visual interest and clear functionality.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and tech-infused feel. Body text font: 'Inter' (sans-serif) for excellent readability and a neutral, professional appearance.
- Utilize simple, functional line icons for actions like add, edit, delete, and search, ensuring clarity and ease of understanding across the interface.
- A clean and responsive layout that optimally utilizes screen space, featuring a clear song list alongside an intuitive form for detailed song management on desktop, adapting seamlessly for mobile devices.
- Subtle animations and transitions, such as slight fades or shifts, to provide immediate feedback on user interactions like form submissions, saves, or deletions without distracting from the content.
